GNU Midnight Commander (or `mc`) is a visual, dual-pane file manager. It is released under the GNU General Public License and therefore qualifies as Free Software.

Midnight Commander is a feature-rich, full-screen, text-mode application that allows you to copy, move, and delete files and entire directory trees, search for files, and execute commands in the subshell. Internal viewer, editor and diff viewer are included.

## Installation

The easiest way to install `mc` is to use your system package manager:

[Debian / Ubuntu](https://midnight-commander.org/#__tabbed_1_1)[Fedora / Red Hat](https://midnight-commander.org/#__tabbed_1_2)[FreeBSD](https://midnight-commander.org/#__tabbed_1_3)[OpenBSD](https://midnight-commander.org/#__tabbed_1_4)[macOS](https://midnight-commander.org/#__tabbed_1_5)

`# apt-get install mc`

Our [source releases](https://ftp.osuosl.org/pub/midnightcommander/?C=N;O=D) are kindly mirrored by OSU OSL. Our [canonical repository](https://github.com/MidnightCommander/mc) is hosted on GitHub. See the [Source code](https://midnight-commander.org/source-code/) page for details.

## Documentation

The primary way to learn about `mc` is to use the context-sensitive online help available via F1.

We also have extensive manual pages, which are the primary source of official documentation:

[mc](https://midnight-commander.org/#__tabbed_2_1)[mcedit](https://midnight-commander.org/#__tabbed_2_2)[mcview](https://midnight-commander.org/#__tabbed_2_3)[mcdiff](https://midnight-commander.org/#__tabbed_2_4)

`$ man mc`

... or read the latest [development version](https://source.midnight-commander.org/man/mc.html) online.
